## 画面詳細

### トップページ（/）

#### ヘッダー
- サイト名「魚偏漢字」大きく表示
- キャッチコピー「居酒屋メニューも怖くない」

#### タブ
- 「読みから探す」「形から探す」の2タブ
- `src/components/Tabs.tsx`

#### 読みから探すタブ
- テキスト入力（ひらがな・カタカナ対応）
- 入力内容に部分一致する漢字をカード一覧で表示
- `src/components/ReadingSearch.tsx`

#### 形から探すタブ
- 左側に「魚」を大きく表示
- 右側に「つくりを選ぶ」ボタン
- ボタンクリックで TsukuriModal を開く
- つくり選択後、選んだつくりと該当漢字のカードを表示
- `src/components/ShapeSearch.tsx`

#### TsukuriModal
- すべての旁（つくり）をグリッド表示
- 選択すると Modal を閉じ、ShapeSearch に結果を返す
- `src/components/TsukuriModal.tsx`

#### 漢字カード
- 漢字（大）・訓読み・音読み・意味を表示
- `src/components/KanjiCard.tsx`

---

## 機能仕様

### 読みから探す

| 機能 | 説明 |
|------|------|
| 入力 | ひらがな・カタカナどちらでも検索可 |
| マッチ条件 | 訓読み・音読みへの部分一致 |
| 空入力 | 全件表示 |

### 形から探す

| 機能 | 説明 |
|------|------|
| つくり選択 | モーダル内のボタンをクリックして選ぶ |
| 結果表示 | 選択したつくりに一致する漢字カードを表示 |
| リセット | 「つくりを選ぶ」ボタン再クリックで別のつくりに変更可 |

## データ

- ファイル：`src/data/kanji.json`
- 件数：約30字
- フィールド：`kanji`, `kun`（訓読み配列）, `on`（音読み配列）, `meaning`, `tsukuri`（旁の文字）
