## 画面詳細

### トップページ（/）

#### ヘッダー（sticky）
- サイト名「魚へんの漢字 検索サイト」
- タブ「読みから探す」「形から探す」

#### タブ
- 「読みから探す」「形から探す」の2タブ
- page.tsx 内で state 管理（`activeTab`）

#### 読みから探すタブ
- テキスト入力（ひらがな・カタカナ相互変換対応）
- 空入力時：全件表示
- 入力時：訓読み・音読みへの部分一致、または漢字そのものへの一致でフィルタ
- `app/components/ReadingSearch.tsx`

#### 形から探すタブ
- 構造タイプ選択トグル：「魚が左」（左右構造）/ 「魚が下」（上下構造）
- 構造プレビューボックス（240×240px）：
  - 未選択時：魚へん（SVGで切り抜き表示）＋「？」ボタン
  - 「？」クリックで TsukuriModal を開く
  - つくり選択後：完成形の漢字を大きく（8rem）アクセントカラーで表示。タップでモーダル再オープン
- 構造タイプ変更時：つくり選択をリセット
- `app/components/ShapeSearch.tsx`

#### TsukuriModal
- 選択中の構造タイプに対応するつくりのみ表示（構造をまたがない）
- テキスト入力でつくりをフィルタ可能
- 4列グリッド表示
- 特殊文字（𦥯=學の上部分、鰖の右部分）はSVGで表示
- 選択すると Modal を閉じ、ShapeSearch に返す
- `app/components/TsukuriModal.tsx`

#### 漢字カード
- 漢字（大）・訓読み・音読み・意味を表示
- クリックで詳細モーダルを表示（読み・意味・例など）
- `app/components/KanjiCard.tsx`

## 機能仕様

### 読みから探す

| 機能 | 説明 |
|------|------|
| 入力 | ひらがな・カタカナどちらでも検索可（相互変換して比較） |
| マッチ条件 | 訓読み・音読みへの部分一致、または漢字そのものへの完全一致 |
| 空入力 | 全件表示 |

### 形から探す

| 機能 | 説明 |
|------|------|
| 構造タイプ選択 | 「魚が左」（左右）/ 「魚が下」（上下）を切り替え |
| つくり選択 | モーダル内のボタンをクリックして選ぶ |
| 選択後表示 | 240×240ボックスに完成形の漢字を大きく表示。タップで選び直し可 |
| 結果グリッド | 選択した構造タイプに一致する全漢字カードを表示（未選択時も全件表示） |
| リセット | 構造タイプを切り替えるとつくり選択がリセットされる |

## データ

- ファイル：`app/data/kanji.json`
- フィールド：`kanji`, `kun`（訓読み配列）, `on`（音読み配列）, `meaning`, `tsukuri`（旁の文字配列）, `structure`（"左右" | "上下"）
